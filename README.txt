
Please read the documents in the "docs" folder.

The minimum components are in the folder "SNP_Viewer_components".

The folder "simple_sample_webapp" contains the simplest possible web app.
It is designed as a quick self-contained demonstration.

The folder "example_read_fasta_from_server" contains a web app that reads
standard "newick" files and fasta files.
      Please see the file example_read_fasta_from_server.txt in the
      "docs" folder for more info.


If you are loading data and get a message that the SNP viewer failed to load,
check the console in the browser for the error.
